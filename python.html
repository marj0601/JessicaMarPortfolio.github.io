Python Unit 1

Description:

Learn the basics of Python through the Python Shell. We also learned about getting the computer to do specific math functions using the Python Shell and creating methods. In addition, we learned how to use classes, set up parameters and learn how to use pygame. In pygame, we learned about user interaction and based of a specific key that the user pressed, the image in pygame would move.

Concepts Learned:

  • Python basics - Learned how to print in the python shell, escape sequences, variables, user input, input with numbers, math operators
  • Escape sequences - I used this to get numbers of letters in a word to move to the next line and how to make quotations or backslashes appear when printed
  • Variables - Used this to help control values that change
  • Boolean statements - Helps determine what the computer will do based on a specific situation. The statement is a true or false, so the answer is either true or false
  • Strings - Helped me write numbers in the python shell after user input
  • Classes - Gives the class attributes to be able to receive input from other objects or the user

Python Unit 2

Description:

Use if statements to respond to user input. If the user pressed a key, then in pygame, the person would move up, down, right, and left. We also learned about how to collide with other objects using if-elses so that the person wouldn’t be able to go through the wall. In the last lab for this unit, we learned how to allow the person to hop over the wall by using nested if statements.

Concepts Learned:

  • If statements - Used if statements to receive user input. For the person, if the user pressed the up, down, right, left arrows, the person would go up, down, right, and left
  • Python methods - I created methods for the person in collision so that the person couldn’t go through the wall
  • If-else statements - Helped with detecting collision so that the person couldn’t go through the wall
  • Nested if statements - I used this by creating an upCounter variable that after the person got to a certain height, he would start going down, without having to program a down arrow for the person to go down

Python Final Project

Description:

For my python final project I created a maze and the elf would have to reach to cookie before the time ran out to win. The elf would start at the top left corner at ( 0, 0 ) and make it to the cookie. If the elf doesn’t reach the cookie before time runs out, then the game is over. If the elf gets the cupcake, then the time is increased by 3.

Concepts Learned:

  • Lists - I used lists like a matrix to create a series of 1’s and 0’s to create the maze
  • Sizing - I had to resize my elf to fit between the walls of the maze
  • Level - I created a variable to control the level so that when a key was pressed it would start at level 1
  • Start screen - I created a start screen so that when 1 was pressed it would go to level 1 and when 2 was pressed it would go to level 2
