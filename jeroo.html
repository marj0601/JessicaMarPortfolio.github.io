Jeroo Unit 1

Description:

Use jeroo to make jeroos move across island with the basic functions. We had to create jeroos, and program them to complete a task with the functions: hop, pick, plant, and toss. During jeroo unit 1 we learned about the different things jeroos could do.

Concepts Learned:

  • Creating jeroos - learned how to create a jeroo and make it appear on the screen
  • Using hop, pick, and plant - learned how to use hop(), pick(), and plant(), so the jeroo could do different things on the island
  • Multiple jeroos - used multiple jeroos to complete a task instead of repeating code many, many times to complete a task

Jeroo Unit 2

Description:

Use jeroo methods to make the main method more organized and simpler. We created jeroo methods to help repeat less amount of code over and over again. Creating jeroo methods also helped finish the labs easier because the method could be applied to multiple jeroos.

Concepts Learned:

  • Jeroo methods - Used methods to organize code and help the program much quicker. Methods help with shorter code in the main method and can be used in multiple jeroos.
  • Creating shapes - Used methods to create shapes that have the same pattern. Created squares and boxes.
  • Turning - Used the turn(LEFT); or turn(RIGHT); functions to make the jeroo turn left or right to reach a certain destination.

Jeroo Unit 3

Description:

Used loops to keep the jeroo doing a certain function until that statement is false. The while loop was used a lot to keep a jeroo hopping while the statement was true. We made spirals and pinballs.

  • Using the while loop - kept going on unless the statement inside the while loop was false
  • Using boolean conditions - this is the code that is put into the while loop so the computer knows that if the boolean condition is true, it will continue running that piece of code
  • Direction - the direction that a jeroo faces helps determine was is ahead. We used the isFlower(AHEAD); and isWater(AHEAD); inside the while loop
