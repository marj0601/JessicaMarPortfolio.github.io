Java Unit 0

Description:

We learned how the basics of java such as how to print in java, how to create classes in java, and how to use escape characters. We did 3 labs in Unit 0. In the introduction to java, we learned about how to print out words, how to print words on new lines, and how to get the computer to do the basic math functions (add, subtract, multiply, and divide). In labs 1 and 2, we learned how to print characters on the next line using System.out.println. In the input lab, we learned about integers, doubles, floats, and shorts. We inputted numbers and the computer would convert the numbers into integers, doubles, floats, and shorts.

Concepts Learned:

  • Printing in java - We learned how to print using System.out.print() and how to print characters on the next line using System.out.print(\n \n) and System.out.println()
  • Escape characters - Escape characters helped me print characters that serve other purposes. I used escaped characters in the introduction to java lab with \n to print characters on the next line
  • Integers, doubles, floats, and shorts - We learned how to print different types of values with integers, doubles, floats, and shorts. I used these in the input lab

Java Assignment

Description:

For the java assignment we created our own mini adventure and based on the user’s choices, there would be different outcomes. For my adventure, the person is in a haunted house on Halloween and there are different choices the user can input for good, okay, and bad results. I created an upstairs, and downstairs option and more options inside of that.

Concepts Learned:

  • Keyboard.nextLine - We learned how to make the adventure flow based on what the user inputted with keyboard.nextLine. I used this a lot to make my adventure flow better
  • Ignore cases in java - I used this so it doesn’t matter if the user types uppercase or lowercase, the program will still run
  • If and else if - I used a lot of if and else if statements to create different options for the user to choose from
  • Java comments - Java comments really helped me to organize my code so it was easier to find and edit another stage
